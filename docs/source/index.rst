Welcome to TinyDiff's documentation!
========================================

**TinyDiff** is a minimal implementation of diffusion models, including the full implementation of the original latent diffusion models.

Check out the :doc:`usage` section for further information, including :ref:`installation` of the project.

.. note::

   This project is under active development.
